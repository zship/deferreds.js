Deferreds.js

Deferreds.js is a utility library centered around working with Deferred objects. The main goal is to do for Deferred objects what async.js does for Node-style asynchronous functions—that is, provide many common higher-order functions (map, reduce, filter, etc.) accepting potentially-asynchronous Deferred objects as arguments. These are located in the Deferreds module.
